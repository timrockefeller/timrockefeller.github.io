数组定义

数组这个东西十分有趣，它的定义是相同类型的一组数据集变量。
在Java中，它能够这样被定义：

c++中了解到，数组的本质是指针，而Java里面并没有指针的强类型概念，但分配空间的规则仍旧是一样的。即当定义一个数组时需要：

* 数据类型（int）
* 数组名称（a）
* 数组 ~长度~ （n=5）

我们在使用数组时，用下标来获取、赋予内容，如a[0]=a[1]===a[n-4]?a[4]:a[0];
