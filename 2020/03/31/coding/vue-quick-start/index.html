列几个坑点

computed不被侦听

在计算属性(computed porperties)配合getter、setter时，getter内部变量发生变化时不会触发计算。即使是dom绑定了这个值也不会被侦听，而是需要主动调用更新。

这时候其实可以用console.log去取个值。

实际项目中我甚至加了些Promise，感觉线程要爆炸，故在本组件中加了个信号量。
